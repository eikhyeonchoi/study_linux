# vi
## 명령모드(진입 시)
```
i: 입력모드 전환(입력모드에서 ESC누를 시 다시 명령모드)
x: 커서위치의 문자 1글자 삭제(5x면 5개)
dw: 커서위치의 단어 삭제
dd: 커서위치의 한줄 삭제
u: 직전 명령 취소
yy: ctrl + c
p: ctrl + v
0(숫자): home
$: end
G: 파일 끝 이동
```

## 입력모드
```
말그대로 입력모드
입력모드에서 ESC누를 시 명령모드로 전환
```

## 행 모드
```
명령모드에서 ":" 누를 시 행모드로 전환
행 모드에서 ESC누를 시 다시 명령모드로 전환
wq: 저장 후 종료
wq!: 저장 후 종료 강제
set nu: 라인번호 출력
set nonu: 라인번호 출력 취소
/string: 커서위치부터 앞쪽으로 string 탐색
?string: 커서위치부터 뒷으로 string 탐색
```
